## Descripcion


Este repositorio tiene como función recopilar los archivos que han sido generados y/utilizados durante el transcurso de la asignatura Herramientas Computacionales para la Astroinformática, desde ejercicios realizados en clase en conjunto con el profesor, como la edición de los mismos. Cuenta con diversos scripts tanto en C y Python en función de lo aprendido en clases, además de los archivos de texto que contienen mis notas personales de cada clase, clasificadas por temática.

La actualización de este respositorio es constante.
